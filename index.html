1.Greatest Common Divisor

The purpose of this program is to find the greatest common divisor of given two variables.

Concepts Used:

  1. Airthematic Operator(%)
  2. Comparison operator(!=)
  3. while loop

Python Code


a=int(input("enter value of a ="))
b=int(input("enter value of b="))
rem=a % b
while rem != 0:
    a=b
    b=rem
    rem=a % b
print("Greatest common divisor of a and b:",b)

4.To find the maximum number in a list

The purpose of this program is to find the maximum number in a given list

Concepts Used:

  1. for loop
  2. comparison operator(==)
  3. append
  4. if-condition

Python Code


n=[]
a=int(input("enter the total no of items:"))
for i in range (1,a+1):
    value=int(input(f"enter value {i}:"))
    n.append(value)

maximum=n[0]
    
for i in n:
    if i> maximum:
        maximum =i
print("maximum value is:",maximum)
